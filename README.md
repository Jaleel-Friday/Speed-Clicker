# Speed-Clicker
A simple Java clicker game built using Swing and timers.
Clicker Game – Java Swing

A simple reaction-based clicker game built in Java using Swing and Timers.
The player chooses a difficulty, presses start, and must click the moving button as many times as possible before the timer runs out.

Features:

Difficulty selection (affects object speed)

Moving “Click Me!” button

Countdown game timer

Randomized colors

Score counter

Clean Java Swing GUI

Uses javax.swing.Timer for game logic

Technologies Used:

Java

Swing GUI

OOP structure

Event-driven programming

How to Run:

Download the project files

Open the project in any Java-capable IDE (IntelliJ, Eclipse, VS Code with Java extensions, etc.)

Compile and run ClickerTest.java

What I Learned:

Working with Swing components

Using timers for animation and game loops

Handling user input

Structuring basic GUI games with Java
